# Simple Banking System

## Project Description

Simple Banking System is a console-based Java application developed using Object-Oriented Programming (OOP) concepts. The system allows users to create bank accounts, deposit money, withdraw money, check account balances, and view account details securely using a PIN-based authentication system.

## Features

* Create Multiple Bank Accounts
* Deposit Money
* Withdraw Money
* Check Account Balance
* Display Account Details
* PIN-Based Security
* Duplicate Account Number Validation
* Invalid Amount Validation
* Insufficient Balance Handling

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class

## OOP Concepts Implemented

* Classes and Objects
* Constructors
* Encapsulation
* Getter Methods
* Method Calling

## Project Structure

SimpleBankingSystem/

* BankingSystem.java
* BankAccount.java

## How to Run

1. Compile the source files:

javac *.java

2. Run the application:

java BankingSystem
